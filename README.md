# Bash DEB build.

Bash-script for build deb package:
1. build:
 *  sudo bash-deb-build
2. unpack:
 *  bash-deb-unpack package.deb

## Depends

dpkg, grep, tar, gzip, bzip2, lzma, coreutils, binutils
